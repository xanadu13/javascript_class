02_데이터 타입(data type) p.468

자바스크립트에서 사용하는 데이터 유형에 대해 알아보기

  • 문자데이터형 (string)
  • 숫자데이터형 (number)
  • 부울형 (boolean) - true(확인), false(취소)
  • 찾을 수 없는 데이터형 (undefined)
  • 값이 없는 데이터 (null)
  • 배열형 데이터 (array)
  • 함수형 데이터 (function)
  • 객체(object) - document, window, alert, confirm ...

typeof - typeof 연산자는 사용자가 입력한 데이터가 어떤 종류의 형식인지를 알려주는 연산자이며, 변수 앞에 Number, String 등을 작성하면 실제 데이터 형식을 변환할 수 있다.
