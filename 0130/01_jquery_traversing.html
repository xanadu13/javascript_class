01jquery DOM_traversing(문서객체 탐색)

문서객체모델(DOM) - 제이쿼리 traversing (탐색) 메서드는 기존의 선택자보다 좀 더 html문서의 요소들을 정확하게 해주는 메서드이다.

css 선택자 부모아래 자식요소 선택은 '부모>자손', '부모 자손', 형제요소 선택시 'A + B', 'A ~ B' 방법을 사용한다.

제이쿼리에서는 html문서의 구조에서 부모요소를 찾거나 부모의 형제들, 부모아래 자식요소들, 자식요소에서 부모의 부모들을 찾을 수 있는 메서드들이다.

1. 부모요소 찾기

  • parent(); - 선택한 요소 바로 위에 위치한 부모요소를 선택한다.
  • parents(); - 선택한 요소의 위에 위치한 모든 부모요소를 선택한다.
  • parentsUntil(); - 선택한 요소에서부터 지정한 요소전까지 부모요소를 선택한다.

2. 자식요소 찾기

  • children(); - 선택한 요소의 아래에서 자식요소를 선택
  • find(); - 사용자가 선택한 요서의 아래에서 지정한 이름과 일치하는 요소 선택

3. 형제요소(sibling) 찾기

siblings 메서드는 사용자가 선택한 요소를 기준으로 형제관계에 있는 (동일한) 요소에서 이전, 다음요소들을 선택할 때 사용

  • siblings() - 현재요소의 형제관계인 요소를 모두 선택한다. (자신은 제외)
  • next() - 현재요소의 다음에 오는 요소를 선택
  • nextAll() - 현재요소의 다음에 있는 모든 요소를 선택
  • nextUntil() - 현재요소의 다음에서부터 지정한 요소 이전까지를 선택
  • prev() - 현재요소의 이전에 있는 요소를 선택
  • prevAll() - 현재요소의 이전에 있는 모든 요소를 선택
  • prevUntil() - 현재요소의 이전에 있는 요소부터 지정한 요소 전까지를 선택

4. 탐색(traversing) 메서드

html문서의 구조에서 사용자가 선택한 요소를 정확하게 찾아낼 수 있는 메소드중의 하나이다.

  • first() - 형제관계 요소들중에서 첫번째 요소를 선택
  • last() - 형제관계 요소들 중에서 마지막번째 요소를 선택
  • eq(n) - 인덱스값의 n번째에 해당하는 요소를 선택
  • filter() - 사용자가 지정한 이름에 맞는 요소를 선택
  • not() - 사용자가 지정한 이름이 아닌 요소를 선택
